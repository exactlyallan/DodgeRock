# Skill: Procedural Visuals & Parameterized Drawing
- **Canvas-First:** Use `PixelArt.ts` to draw via code rather than loading external PNGs.
- **DNA Strings:** Design entities to be "parameterized." (e.g., `drawPlayer({ horns: true, color: 0xFF0000 })`).
- **The Designer:** When building UI for the "Sprite Designer," map sliders directly to these function parameters.
